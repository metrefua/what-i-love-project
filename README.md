# what-i-love-project
Overview
This project is a personal and creative website dedicated to the beauty and meaning of sunsets. It blends web development practice with poetic storytelling, serving both as a portfolio piece and a heartfelt expression. Visitors can explore galleries, reflections, and interactive elements that invite them to share in the calm and wonder of twilight.
#Features
- Homepage with welcoming introduction, personal meaning, navigation hints, and a closing quote
- About page explaining the inspiration, creator’s journey, and symbolism of sunsets
- Gallery showcasing sunset images in different moods and colors
- Reflections page with writings inspired by twilight
- Contact form where visitors can share their favorite sunset moments and preferences
- Educational sections with scientific, cultural, and fun facts about sunsets, including comparison tables
#Technologies Used
- HTML5 for structure and semantic layout
- CSS3 for styling and thematic colors inspired by sunsets
#Project Goals
- Practice and improve web development skills
- Create a website that balances technical rigor with poetic warmth
- Share personal meaning and cultural insights about sunsets
- Build a portfolio piece that reflects both creativity and coding ability
#How to Run
- Clone or download the project files
- Open index.html in any modern web browser
- Navigate through the pages using the links provided
